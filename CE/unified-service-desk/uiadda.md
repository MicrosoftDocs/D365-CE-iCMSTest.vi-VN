# <a name="uiadda-in-unified-service-desk"></a>UIADDA in Unified Service Desk
Bộ chuyển đổi định hướng dữ liệu Tự động hóa UI (UIADDA) cung cấp quyền truy cập vào [!INCLUDE[pn_ms_Windows_Presentation_Foundation](../includes/pn-ms-windows-presentation-foundation.md)] và ứng dụng [!INCLUDE[pn_MS_Silverlight_full](../includes/pn-ms-silverlight-full.md)]. DDA này cũng hỗ trợ ứng dụng web và ứng dụng dựa trên [!INCLUDE[pn_ms_Windows_short](../includes/pn-ms-windows-short.md)]. UIADDA tận dụng API Tự động hóa Giao diện Người dùng (UI). Liên kết cho UIADDA có thể được tạo bởi Trình kiểm tra [!INCLUDE[pn_uii_acronym](../includes/pn-uii-acronym.md)].
